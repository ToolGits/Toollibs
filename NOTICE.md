Toollibs Community Branch Deprecation Notice

Date: 14/06/2026

The "community" branch has entered Legacy status and is no longer actively maintained.

## Why?

Over time, the main Toollibs project evolved significantly and the "community" branch no longer reflects the current architecture, features, and development workflow of Toollibs.

The branch now serves primarily as a historical archive containing:

- Legacy documentation
- Historical Makefile templates
- Early project experiments
- Development history

Replacement

The "community" branch is being replaced by the new:

"nightly"

branch.

Purpose of the Nightly Branch

The "nightly" branch will be used for:

- Experimental features
- Early implementations
- New Toollibs modules
- Platform testing
- Unstable development builds
- Future version development

## Branch Status

Branch | Status

main | Stable / Official

nightly | Experimental / Active

community | Legacy / Archived

Legacy Preservation

The "community" branch will remain available for historical and archival purposes unless otherwise stated.
