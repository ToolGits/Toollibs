▶ v3.0 "Harmonic Duality" LEGACY
  • Major expansion and refinement of Toollibs stable ecosystem
  • Audio module elevated as central system focus of the framework
  • Audio System:
  • Unified multi-format playback pipeline (WAV / OGG / MP3)
  • Improved decoder integration (SDL2 + libsndfile + stb_vorbis + minimp3)
  • More stable cross-platform audio behavior across Linux systems
  • Optimized memory handling for decoded audio buffers
  • Improved playback consistency across backend layers
  • Audio Player Android:
  • Native Android audio implementation introduced
  • ToollibsAudio.java bridge layer for JNI communication
  • audio_backend.cpp unified backend abstraction layer
  • audio_android.cpp Android-specific audio pipeline
  • Support for aarch64 mobile architecture (ARM64)
  • NDK integration for native compilation and performance optimization
  • Hybrid audio system (SDL2 + OpenSL ES readiness layer)
  • Improved latency handling for mobile playback
  • Cross-platform audio parity improvements (Linux ↔ Android)
  • Website Layer:
  • Changelog UI upgraded to fully animated expandable system
  • Improved visual interaction (glow, transitions, motion layering)
  • Better readability and version navigation experience
  • Performance improvements in card rendering and toggle animation
