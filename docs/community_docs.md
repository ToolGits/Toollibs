# Toollibs Community Branch Documentation

This document explains the purpose of the `community` branch in Toollibs, its rules, limitations, and philosophy.

---

## 🌍 What is the community branch?

The `community` branch is a public experimental space for Toollibs.

It is NOT the official version of the project.

It is used for:
- experiments
- prototypes
- community ideas
- testing new systems
- alternative implementations

Anything here may be changed, removed, or rewritten in the future.

---

## ⚠️ Important rules

By contributing to the `community` branch, you agree that:

- Features may be deleted without warning
- Code may be rewritten or replaced
- Nothing here is guaranteed to become official
- Breaking changes are allowed at any time

This branch is a sandbox for innovation.

---

## 🧠 Official vs Community

- `main` → official, stable version of Toollibs
- `community` → experimental, unstable, community-driven

Only `main` represents the original and stable project.

The goal of the community branch is:
