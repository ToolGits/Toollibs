About

Toollibs is a modular C++ framework designed for reusable components, system utilities, runtime tools and scalable architecture.

Core Modules

  • core — logging + runtime system
  • math — vector + math engine
  • graphics — rendering utilities
  • input — device input system
  • plugins — extensibility layer

Toollibs Ecosystem

  • ToollibsServer — backend API system
  • Toollibs Deploy System — automated build distributor
  • Toollibs-workstation — development environment (Debian)
  • cpu_info / gpu_info — Linux hardware tools
  • Website — documentation + downloads + changelog
