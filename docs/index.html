About Toollibs

Toollibs is a modular open-source C++ framework library designed for developers who need scalable, reusable and lightweight systems.

Main Modules

  • core: logging, basic utilities, and system helpers
  • graphics: simple rendering and pixel manipulation
  • input: keyboard, mouse, and controller handling
  • audio: basic sound and audio system
  • fs: file system utilities
  • math: mathematical helpers and structures
  • plugins: modular extension system

Community Branch

Toollibs includes a dedicated experimental community branch for prototypes, testing, new ideas and alternative implementations.

Project Status

Toollibs is currently in active development. The main branch contains the official stable structure.
