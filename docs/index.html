About

Toollibs is a modular C++ framework designed to provide reusable components, system utilities, plugin-based architecture and development tools for scalable projects.

Core Modules

  • core — logging system + runtime control
  • math — vector and math utilities
  • graphics — rendering helpers
  • input — input abstraction layer
  • fs — filesystem utilities
  • plugins — extensibility system

Toollibs Ecosystem

  • ToollibsServer — backend/API system
  • Deploy System — automated build & distribution pipeline
  • Toollibs Workstation — Debian-based development environment
  • cpu_info / gpu_info — Linux hardware inspection tools
  • Web Platform — documentation, downloads and changelog system

Version System
