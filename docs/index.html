Toollibs

Open-source modular C++ framework library for tools, utilities and scalable systems.

Lightweight · Modular · Community-driven

GitHub Repository

About

Toollibs is a modular open-source C++ framework library designed to provide reusable, lightweight and scalable components for developers.

Modules

  • core — logging and core utilities
  • graphics — rendering and pixel tools
  • input — keyboard and mouse handling
  • audio — basic audio utilities
  • fs — file system helpers
  • math — math utilities and structures
  • plugins — extensibility system

Project Status

Toollibs is currently in active development. The main branch represents the current stable structure.

Community

Contributions, ideas and experiments are welcome through the community branch and GitHub repository.
