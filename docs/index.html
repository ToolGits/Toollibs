About Toollibs

Toollibs is a modular open-source C++ framework designed for reusable components, system utilities and scalable development.

v2.0 Forgelibs

First stable release of the framework introducing plugins, improved logging system and multi-architecture support.

Core Modules

  • core — logging and runtime system
  • graphics — rendering utilities
  • input — device input handling
  • math — vector and math utilities
  • plugins — extensibility system

Project Status

Toollibs is now in a stable release state (v2.0 Forgelibs). Future updates will focus on improvements and plugins.
