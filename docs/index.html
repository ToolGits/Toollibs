Project Status

Toollibs is under active development with official support for Linux, Windows and Android. The ecosystem includes modular libraries, system utilities, automated deployment and a continuously evolving documentation platform.

About

Toollibs is an open-source modular C++ framework ecosystem designed for building reusable libraries, development tools and cross-platform applications. It focuses on simplicity, scalability and native performance across supported platforms.

Version System

Toollibs uses a codename-based versioning system introduced in the v2.0 Forgelibs version, generations such as v1.x, v2.x and v3.x, and everything is recorded in the Changelog.
