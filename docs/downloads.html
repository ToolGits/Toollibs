Build System

Toollibs uses an automated build and deployment workflow to compile, organize and distribute binaries across supported platforms.

  • Multi-architecture compilation
  • Automated binary deployment
  • Generated download index
  • Linux, Windows and Android support

Supported Architectures

  • x86_64 — 64-bit desktops and laptops
  • x86 — 32-bit systems
  • aarch64 — ARM 64-bit devices
  • armv7l — ARM 32-bit legacy devices
  • armv6l — ARM 32-bit legacy devices (oldest)
  • Windows builds available (x86_64 only)
  • Automatic architecture-based download selection
